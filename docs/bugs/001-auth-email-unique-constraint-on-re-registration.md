# 회원탈퇴 후 동일 이메일 재가입 실패

## 상태: Open

## 재현 경로

1. 회원가입 (이메일: example@test.com)
2. 로그인
3. 회원탈퇴
4. 동일 이메일(example@test.com)로 재 회원가입 시도

## 증상

- 회원탈퇴 후 동일 이메일로 재가입 시 실패
- `auth` 스키마의 `users` 테이블 `email` 컬럼의 unique 제약 조건 위반 오류 발생

## 원인

- 회원탈퇴 처리 시 `auth.users` 테이블에서 해당 사용자 레코드가 완전히 삭제되지 않거나, soft delete 처리되어 email unique 제약 조건에 여전히 걸림

## 영향

- 탈퇴한 사용자가 동일 이메일로 서비스 재가입 불가
