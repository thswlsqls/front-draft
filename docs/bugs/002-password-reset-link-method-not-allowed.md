# 비밀번호 재설정 링크 클릭 시 METHOD_NOT_ALLOWED 오류

## 상태: Open

## 재현 경로

1. 회원가입
2. 이메일 인증
3. 로그인
4. 로그아웃
5. 비밀번호 재설정 요청
6. 이메일로 발급된 재설정 링크 클릭

## 증상

- 비밀번호 재설정 이메일의 링크를 클릭하면 아래 오류 응답 반환:

```json
{
  "code": "4050",
  "messageCode": {
    "code": "METHOD_NOT_ALLOWED",
    "text": "허용되지 않는 HTTP 메서드입니다."
  }
}
```

## 원인

- 비밀번호 재설정 링크의 HTTP 메서드가 서버에서 허용되지 않는 메서드로 요청되고 있을 가능성 (예: GET으로 요청되어야 할 곳에 POST, 또는 그 반대)
- 이메일 템플릿의 링크 URL 또는 리다이렉트 경로 설정 문제 가능성

## 영향

- 로그아웃 상태에서 비밀번호 재설정 불가
